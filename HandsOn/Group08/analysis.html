def plot_corr(df, width = 15, height = 14):
    corr_matrix = df.corr()
    f = plt.figure(figsize=(width, height))
    plt.matshow(corr_matrix, fignum=f.number)
    plt.xticks(range(corr_matrix.shape[1]),corr_matrix.columns, fontsize=12, rotation=45)
    plt.yticks(range(corr_matrix.shape[1]), corr_matrix.columns, fontsize=12)
    plt.colorbar()
    plt.show()

Correlation matrix¶

Below we can see the correlation matrix across all the meaningfull variables in the dataset. We can see that there are high correlations between similar velocities (20-30 is close to 30-40, 30-40 to 40-50 ,etc) both in normal vehicle speeds as well as across 2-wheeled vehicle speeds, which makes sense.
